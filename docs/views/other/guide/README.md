## 序言

富有生活气息而又短小精悍的日剧一直为不少观众喜爱，而同样备受大家关注的还有与剧本身契合度极高的原声歌曲，那些或朗朗上口、或引人深思充满质感的日剧主题曲，着实为剧集本身添色不少。今天我们来历数一下近几年那些颇有人气的日剧主题曲吧。

## Lemon——Unnatural主题曲
![](asset/Unnatural.jpg) 

近期人气医学悬疑剧《Unnatural（非正常死亡）》正式完结，对于该剧的剧情以及演员表现力绝大多数的影迷都给予了肯定的好评。而由米津玄师创作并演唱的该剧主题曲《Lemon》也同样给大家留下了深刻的印象，每当耐人寻味的真相被揭晓之时响起的旋律让大家直呼虐心。
![](asset/Unnaturalshiyuanlimei.gif)

## TOKYO GIRL——东京白日梦女 主题曲
![](asset/dongjingbairimengnv.jpg) 

由日本国宝级女团电音香水（Perfume）演唱日剧《东京白日梦女》的主题曲《TOKYO GIRL》，延续了Perfume一贯的未来感电音风格，搭配上剧情一起食用，将片中东京单身大龄女子的形象展现得一览无余。

## 成人法则——四重奏 主题曲
![](asset/sichongzou.jpg)

2017年的日剧高分神作《四重奏》，主题曲《成人法则》由稚名林檎创作，四位主演演员松隆子、满岛光、高桥一生、松田龙平共同演唱。《成人法则》有着专属椎名林檎式的独特质感，精美的场景、四位主演传神的演绎，精准的击中了每一位观者的心。
